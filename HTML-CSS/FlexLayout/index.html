What is FlexBox?

Flexbox - a whole module that contains a set of properties and their values for both the parent flex-container and its children flex-elements.

The main advantage of flexible FLEX markup is the ability to change the width/height space around and the order of the flex-elements in order to best fill the available space flex-container.
